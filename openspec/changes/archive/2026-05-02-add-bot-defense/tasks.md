## 3. Engine: DangerAwarePickDiscard

- [x] 3.1 Update Bot Decision Strategy — danger-aware discard score function. Per the decision "danger penalty constant k is chosen to dominate isolation differences", add `Bot.DangerAwarePickDiscard(hand []tile.Tile, danger map[uint8]int) int` to `internal/game/bot.go`. When `danger` is nil or empty, return the result of the existing `b.PickDiscard(hand)`. Otherwise iterate `i := 0..len(hand)-1`, compute `score = isolationScore(i, hand[i], hand) - 2000 * danger[hand[i].ID]` (default penalty for unknown IDs is `2 * 2000 = 4000`, treated as if `danger[id]` is 2 when the key is absent). Track the highest score; tiebreak by lowest tile ID. The K=2000 constant is chosen so that any safe tile beats any unsafe tile regardless of isolation difference.

- [x] 3.4 In `internal/game/bot_test.go`, add `TestBotDangerAwarePickDiscardFallsBackToIsolation`: when all tiles are unknown (or danger map is empty), the result equals `bot.PickDiscard(hand)`.
